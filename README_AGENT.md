# 📚 高校思政热点选题助手 · 升级说明与配置指南

您的辅导员思政助手已完成**史诗级架构重构**！现在，它不仅是一个生成选题的工具网页，而是一个具备**自主决策决策流**、**Google 联网搜索**、**TF-IDF 语义检索知识库**、**专家级自我评估循环**、**一键导出精美 PPT/Word** 以及**每周微信定时推送**的真正「思想政治教育 Agent 智能体」。

## 🌟 新增核心功能介绍

### 1. 🤖 智能体决策舱（Agent Log）
在生成内容时，您将在右侧输出区上方看到一个动态滚动的「命令行日志流」。它公开了 Agent 在后台的所有思考路径：
- 检索知识库匹配度评估；
- 决定是否联网搜索；
- 撰写初稿与调用自我评估算法；
- 扣分项纠错与二次润色过程。

### 2. 🌐 真正的 Google 实时联网检索
引入了专业的 `Serper` 搜索接口支持。填入您的 Serper Key 后，Agent 在生成前会自动将您的「热点词」和思政场景在 Google 中搜索最新政策、新闻、学术文章并加以参考，生成的框架更具时代感和现实政策依据。

### 3. 🧠 基于本地 TF-IDF 的「语义搜索知识库」
您上传的优秀作品不再只是分类随机匹配，页面内置了轻量级信息检索算法：
- 自动提取您当前任务中的关键词；
- 与本地数据库所有案例进行多维余弦相似度（Cosine Similarity）匹配；
- 自动挑选匹配度最高的 2-3 篇案例注入 AI 提示词进行 Few-Shot 临摹。

### 4. ⚖️ 专家级自我评估纠错循环
开启后，生成进入双阶段：
- **阶段一（初稿）**；
- **阶段二（自我校对）**：AI 自动换位成为「高校思政特级评审专家」，从**政治站位、育人实效、青年契合、结构文采**四个维度给初稿打分。如低于 90 分，自动生成修改意见，打回由 AI 主动进行二版重写优化，最终向您交付最优终稿。

### 5. 📥 一键导出精美学术蓝 PPT 与 Word
- **一键导出 Word**：保持干净、完美的字号和标题设计，免除您二次排版的烦恼。
- **一键导出 8 页精美学术 PPT**：采用深度学术蓝（#2c5f8a）和雅致金色（#d4af37）主题。含封面页、活动深度解读页、选题一、二、三方案页、核心递进逻辑图框架页、政策关联与实效页、封底页。

## 🛠️ 三大高阶平台 Key 免费配置指南

为激活全部 Agent 潜能，请按照以下步骤获取服务：

### 第一步：获取 Serper API Key（实现自动搜索）
1. 访问：[https://serper.dev/](https://serper.dev/) 并注册账号（支持 GitHub 或 Google 快捷登录）；
2. 注册即送 **2500次免费搜索额度**，完全够用半年以上；
3. 登录后在 Dashboard 的「API Key」页面复制那串一长串字符；
4. 贴入网页顶部的「Serper API Key」输入框中并保存。

### 第二步：获取 WxPusher 凭证（实现微信推送）
1. 访问 WxPusher 官网：[https://wxpusher.zhimengeng.com/](https://wxpusher.zhimengeng.com/) 并注册/登录；
2. 进入后台，点击「**应用管理**」→「**创建应用**」；
   - 应用名字叫：「思政热点雷达」
   - 关注后提示语：「欢迎订阅您的周度思政选题灵感雷达！」
3. 创建成功后，复制 **`AppToken`**（形如 `AT_xxxx...`，重要，只显示一次！）；
4. 用微信扫码关注您创建的应用。关注后，在后台「**用户管理**」→「**用户列表**」里找到您自己，复制您的 **`UID`**（形如 `UID_xxxx...`）；
5. 将这两个参数填入网页「微信推送设置」面板中，点击「测试微信推送」，您的手机微信应能立刻收到测试消息！

### 第三步：配置 GitHub Actions（实现每周一早上9点定时微信推送）
我们希望您和您的辅导员同事在**每周一早上 9:00 整**都能准时在手机微信上收到本周最新生成的热点，这需要通过您的 GitHub 仓库的定时工作流实现：

1. 打开您的 GitHub 仓库地址：[https://github.com/erica11071972/sizheng-agent](https://github.com/erica11071972/sizheng-agent)；
2. 点击顶部菜单的「**Settings**」标签页；
3. 左侧菜单拉到最下方，找到「**Secrets and variables**」→ 点击「**Actions**」；
4. 点击右侧的「**New repository secret**」按钮，依次添加以下 3 个机密变量：
   - 变量名：`DEEPSEEK_API_KEY`，值：填入您的 DeepSeek API Key（以 `sk-` 开头的那串）；
   - 变量名：`WX_APP_TOKEN`，值：填入您刚才在 WxPusher 申请到的 `AT_` 开头的 AppToken；
   - 变量名：`WX_UIDS`，值：填入您的 WxPusher `UID_`（如果想推送给多位辅导员同事，用英文逗号分隔：`UID_1,UID_2,UID_3`）。

**配置完成后，GitHub 将在每周一早上自动运行生成代码，并直接将最新一期推送到您绑定的所有微信上！**
*(如果您想手动测试：可以在 GitHub 仓库点击「**Actions**」→「**Weekly Sizheng Hotspot Push**」→「**Run workflow**」手动触发测试。)*
